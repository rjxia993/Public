# Hotel Booking Analysis

## Overview
This project analyzes hotel room booking data to uncover trends in reservations, cancellations, and customer behavior. 
The goal was to explore what factors influence whether a booking gets cancelled and identify patterns that could help hotels better manage their inventory.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## What We Did
- Cleaned and preprocessed the raw booking dataset
- Performed exploratory data analysis (EDA) on booking trends
- Visualized cancellation rates across different customer segments
- Identified key factors associated with booking cancellations
- Compared results from different data modeling tools 

## Files
- `hotel_booking.ipynb` — main notebook with full analysis and visualizations
